<9>逆元

定义

如果一个线性同余方程 $ax \equiv 1 \pmod{b}$，则 $x$ 称为 $a\mod{b}$ 的逆元，记作$a^{-1}$。

如何求逆元

扩展欧几里得法
实现

扩展欧几里得法和求解线性同余方程是一个原理，在这里不展开解释。

快速幂法
证明

因为 $ax \equiv 1 \pmod{b}$,
根据费马小定理，有 $ax \equiv a^{b-1} \pmod{b}$,
所以 $x \equiv a^{b-2} \pmod{b}$

然后我们可以用快速幂来求。

实现

注意

  • 快速乘法使用了费马小定理，要求 $b$ 是一个素数；
  • 扩展欧几里得法只要求 $\gcd(a, b) = 1$。
线性求逆元

求出1，2，…，n中每个数关于质数p的逆元。

如果对于每个数进行单次求解，以上两种方法就显得慢了，很有可能超时，所以下面来讲一下如何
线性 O(n) 求逆元。

​首先​​，显然有
$1^{-1} \equiv 1 \pmod{p}$
​​证明​​：
对 $\forall p \in \mathbb{Z}$，$1 \times 1 \equiv 1 \pmod{p}$ 恒成立，故在模 $p$ 下1的逆元是1。

​递归情况​​：
对 $i^{-1}$，令
$k = \left\lfloor \frac{p}{i} \right\rfloor,\quad j = p \bmod i$
则有 $p = ki + j$，即
$ki + j \equiv 0 \pmod{p}$
当 $p$ 为质数时，$j$ 在模 $p$ 意义下的乘法逆元必定存在。在上式两边同时乘 $i^{-1} \times j^{-1}$ 得：
$k j^{-1} + i^{-1} \equiv 0 \pmod{p}$
即
$i^{-1} \equiv -k j^{-1} \pmod{p}$
代入 $j = p \bmod i$ 得：
$i^{-1} \equiv -\left\lfloor \frac{p}{i} \right\rfloor (p \bmod i)^{-1} \pmod{p}$

​迭代实现​​：
注意到 $p \bmod i < i$，假设已求得所有 $j < i$ 的模 $p$ 逆元 $j^{-1}$，可得递推式：
$i^{-1} \equiv \begin{cases} 1 & \text{if } i = 1, \
-\left\lfloor \frac{p}{i} \right\rfloor (p \bmod i)^{-1} & \text{otherwise}.
\end{cases} \pmod{p}$

实现

使用 $p - \left\lfloor\frac{p}{i}\right\rfloor$ 来防止出现负数。

​当 p 不为质数​​：
线性同余方程中指出，如果 i 与 p 不互素时不存在相应的逆元。如果 p 不为质数，则至少有一个 i 的逆元不存在。此时建立在逆推式上的方法不能保证结果正确性。例如当 $p = 8$ 且 $i = 3$ 时，根据递推式，逆元需要从 $inv[p \bmod i]$ 即 $inv[2]$ 计算。而 2 在模 8 意义下不存在逆元，$inv[2]$ 的值是未定义的（准确来讲，递推时 $inv[2]$ 依赖于 $inv[0]$ 的初始值）。因此 3 在模 8 意义下的乘法逆元便无法正确求出。

另外，根据线性求逆元方法的式子：
$$i^{-1} \equiv -k j^{-1} \pmod{p}$$
递归求解 $j^{-1}$，直到 $j = 1$ 返回 1。
中间优化可加记忆化，求 $1,2,\dots,n$ 中所有数逆元的时间复杂度仍是 $O(n)$。
注意：用给定式子递归求单个数逆元的时间复杂度上界为 $O(n^3)$，算法竞赛中更好的方法有扩展欧几里得法和快速幂法。

线性求任意 n 个数的逆元

上述方法只能求 1 到 n 的逆元，若求任意给定 n 个数（$1 \leq a_i < p$）的逆元，方法如下：

  1. 计算 n 个数的前缀积，记为 $s_i$
  2. 使用快速幂或扩展欧几里得法计算 $s_n$ 的逆元 $sv_n$
  3. 因为 $sv_n$ 是 n 个数积的逆元，乘 $a_n$ 抵消后得 $a_1$ 到 $a_{n-1}$ 积的逆元 $sv_{n-1}$
  4. 同理可依次计算出所有 $sv_i$，$a_i^{-1}$ 可用 $s_{i-1} \times sv_i$ 求得

可在 $O(n + \log p)$ 时间内算出 n 个数的逆元。
